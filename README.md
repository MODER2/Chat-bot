# JavaFX Чат-бот (Задание 2)

Учебный чат-бот с графическим интерфейсом на **JavaFX**. Демонстрирует обработку
текста и **регулярные выражения**, хранение истории сообщений и (опционально)
веб-запросы и обращение к LLM-серверу.

## Возможности

- Стартовое окно для ввода имени пользователя, затем основное окно чата.
- Ответы на приветствия («Привет, Бот!»), вопросы («Который час?», «Какое сегодня
  число?»), статистику переписки.
- Команды с параметрами через regex: «умножь 12 на 157», «сложи 2 и 3», «15 / 4».
- История сообщений в оперативной памяти с автором и временем; сохранение в файл
  при выходе и загрузка при запуске.
- Отправка по **Enter** / **Ctrl+Enter** (Shift+Enter — новая строка).
- Своя цветовая палитра, шрифт и иконка приложения.
- Демонстрация веб-запроса («расскажи факт») и обращения к нейросети («ии: …»).

## Быстрый старт

```bash
git clone https://github.com/MODER2/Chat-bot
cd Chat-bot
mvn clean javafx:run
```

Требуется **JDK 17+**. JavaFX подтянется автоматически через Maven.
Подробности — в [`ИНСТРУКЦИЯ.md`](ИНСТРУКЦИЯ.md).

## Архитектура (3 слоя)

| Слой | Пакет | Главные классы |
|------|-------|----------------|
| **Model** (данные) | `model` | `Message`, `ChatHistory`, `Author` |
| **Логика бота** | `bot`, `bot.handler` | `IBot`, `AbstractBot`, `SimpleBot`, `MessageHandler` + обработчики |
| **Хранение** | `persistence` | `HistoryStorage`, `FileHistoryStorage` |
| **UI** | `ui` | `ChatApplication`, `LoginWindow`, `ChatWindow` |

Логика бота не зависит от JavaFX, поэтому её можно переиспользовать (например, в
онлайн-боте). Новые команды добавляются одним классом-наследником `MessageHandler`
без изменения существующего кода.
